# 90 Days Trading Journal 📈

A simple and structured **90 Days Trading Journal** designed to help traders track their trades, analyze performance, and improve trading discipline.

## Overview
This project allows traders to record and review their trades over a **90-day challenge period**. By maintaining a consistent trading journal, traders can identify patterns, control emotions, and improve decision-making.

## Features
- Track **trade entry and exit**
- Record **profit and loss**
- Monitor **risk-reward ratio**
- Add **notes and trading psychology**
- Review **daily trading performance**
- Build discipline with a **90-day challenge**

## Purpose
The goal of this project is to help traders:
- Develop **consistency**
- Improve **risk management**
- Analyze **trading mistakes**
- Track progress over time
